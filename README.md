JUDUL APLIKASI : SISTEM INVENTARIS BARANG

DESKRIPSI      : 
Sistem Inventaris Barang adalah aplikasi web sederhana berbasis 

PHP Native dan MySQL yang digunakan untuk mencatat dan mengelola 

data barang/inventaris kantor secara digital. Aplikasi ini memiliki 

4 halaman utama: Beranda (dashboard ringkasan statistik inventaris 

seperti total jenis barang, kategori, stok menipis, dan total nilai 

inventaris), Tambah Data (form input barang baru), 

Daftar Data (tabel seluruh data barang lengkap dengan fitur pencarian), 

dan Edit Data (form untuk mengubah data yang sudah ada).

Setiap barang otomatis ditandai dengan status stok Aman, Menipis, 

atau Habis sesuai jumlah yang tersisa, sehingga pengguna bisa langsung 

tahu barang mana yang perlu segera diisi ulang. Aplikasi ini 

mengimplementasikan operasi CRUD (Create, Read, Update, Delete) 

penuh ke database MySQL, serta menerapkan konsep dasar PHP seperti 

variabel, percabangan, perulangan, function, include/require, dan 

form processing (GET/POST).
